## 新系统

重构后的新系统位于 `../src/` 目录，主要改进：

1. **配置化**：医院特定规则提取到 YAML 配置文件
2. **模块化**：分类器、解析器、导出器分离
3. **可扩展**：通过继承基类适配新医院
4. **更安全**：重复文件移到回收站而非删除

## 使用新系统

运行项目根目录的 `run.py` 或 `双击运行.bat`
